# 📊 Student Performance Analysis

## 📌 Overview
This project analyzes student performance data using Python to identify key factors influencing academic success. The analysis focuses on gender and parental education level.

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

## 📊 Key Insights
- Gender differences in performance are relatively small  
- Parental education level has a noticeable impact on student scores  

## 📁 Files Included
- student_performance_analysis.html → Interactive version of the analysis  
- student_performance_analysis.pdf → Clean report version
