Project Description¶

This project analyzes student performance data to identify factors that influence academic success. The analysis focuses on gender and parental education level.

import pandas as pd
import numpy as np

# إنشاء بيانات طلاب بشكل مباشر
np.random.seed(42)

data = {
    "gender": np.random.choice(["male", "female"], 100),
    "parental level of education": np.random.choice([
        "high school", "bachelor", "master"
    ], 100),
    "math score": np.random.randint(50, 100, 100),
    "reading score": np.random.randint(50, 100, 100),
    "writing score": np.random.randint(50, 100, 100)
}

df = pd.DataFrame(data)

import matplotlib.pyplot as plt

df.groupby("parental level of education")["math score"].mean().plot(kind="bar")

plt.ylabel("Average Math Score")

Key Insight¶

The analysis shows that students with more highly educated parents tend to perform better academically, especially in math. While gender differences exist, they are relatively small, suggesting that parental education may have a stronger influence on student performance.

Conclusion¶

The analysis revealed that parental education level plays a role in student performance, especially in math. While gender differences exist, they are relatively small. These insights can help educators focus on strategies that support students more effectively.
